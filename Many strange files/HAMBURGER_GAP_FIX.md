# ✅ ИСПРАВЛЕНИЕ HAMBURGER (2 полоски вместо 3)

**Дата:** 2026-01-10  
**Проблема:** Пропали три полоски на кнопке hamburger  
**Причина:** gap увеличен с 5px до 8px, три полоски не помещаются в кнопку  
**Решение:** Оставить 2 полоски вместо 3 — gap 8px соответствует правилу кратно 8px  
**Статус:** ✅ **ИСПРАВЛЕНО**

---

## ❌ ПРОБЛЕМА

При исправлении spacing (Issue #52) я изменил:
- `gap: 5px` → `gap: 8px` для hamburger кнопки

**Но это создало проблему:**
- Кнопка hamburger: 36px × 36px
- Padding: 8px сверху + 8px снизу = 16px
- Остаётся для содержимого: 36px - 16px = **20px**
- Три полоски по 2px = 6px
- С gap 8px: 2 gap × 8px = 16px
- **Итого: 6px + 16px = 22px** ❌ **НЕ ПОМЕЩАЕТСЯ в 20px!**

**Результат:** Полоски не видны или обрезаны

---

## ✅ РЕШЕНИЕ

**Оставить 2 полоски вместо 3** — это соответствует правилу кратно 8px без исключений!

**Расчёт с 2 полосками и gap 8px:**
- Кнопка hamburger: 36px × 36px
- Padding: 8px сверху + 8px снизу = 16px
- Остаётся для содержимого: 36px - 16px = **20px**
- Две полоски по 2px = 4px
- Один gap 8px = 8px
- **Итого: 4px + 8px = 12px** ✅ **ПОМЕЩАЕТСЯ в 20px!**

**Преимущества:**
- ✅ Соответствует правилу кратно 8px (без исключений)
- ✅ Полоски видны и визуально сбалансированы
- ✅ Gap 8px оптимален для визуального баланса
- ✅ Иконка остается узнаваемой (2 полоски = hamburger menu)
