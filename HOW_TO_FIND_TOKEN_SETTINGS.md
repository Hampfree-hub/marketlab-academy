# 🔍 Как найти настройки токенов на GitHub (без ссылки)

**Проблема:** Не знаете где создать токен, если не по прямой ссылке  
**Решение:** Пошаговая инструкция через интерфейс GitHub

---

## 📋 Пошаговая инструкция (с картинками в голове):

### Способ 1: Через ваш профиль (самый простой)

#### Шаг 1: Откройте GitHub
```
https://github.com
```

#### Шаг 2: Нажмите на ваш аватар (правый верхний угол)
- В правом верхнем углу экрана
- Рядом с уведомлениями (колокольчик)
- Нажмите на вашу фотографию/аватар

#### Шаг 3: Выберите "Settings"
- В выпадающем меню
- Нажмите "Settings" (Настройки)

#### Шаг 4: В левом меню найдите "Developer settings"
- Прокрутите левое меню вниз
- В самом низу будет раздел "Access"
- Внутри будет "Developer settings"
- Нажмите на "Developer settings"

#### Шаг 5: Выберите "Personal access tokens"
- В подменю "Developer settings"
- Выберите "Personal access tokens"
- Подраздел: "Tokens (classic)"

#### Шаг 6: Нажмите "Generate new token"
- Кнопка справа вверху
- Выберите "Generate new token (classic)"

---

### Способ 2: Через настройки репозитория (если вы в репозитории)

**Если вы сейчас в репозитории** (как на вашем скриншоте):

#### Шаг 1: В левом меню найдите "Security"
- В настройках репозитория (Settings)
- Раздел "Security"

#### Шаг 2: Но это НЕ то место!
- ❌ "Secrets and variables" — это для репозитория, не для токенов
- ❌ "Deploy keys" — это для SSH ключей

**Вывод:** Для создания токена нужно выйти из репозитория!

---

## 🎯 Правильный путь (самый простой):

### Вариант А: Прямая ссылка (быстрее всего)
```
https://github.com/settings/tokens?type=beta
```

### Вариант Б: Через меню (если не хотите ссылку)

**Путь:**
```
GitHub.com
  → Ваш аватар (правый верхний угол)
    → Settings
      → Developer settings (внизу слева)
        → Personal access tokens
          → Tokens (classic)
            → Generate new token
              → Generate new token (classic)
```

---

```
┌─────────────────────────────────────┐

└─────────────────────────────────────┘
```

## ⚡ Быстрый способ (рекомендую):

**Просто используйте ссылку:**
```
https://github.com/settings/tokens?type=beta
```

**Или:**
```
https://github.com/settings/tokens
```

**Это самый быстрый способ!** ✅

---

## 🔑 После того как найдёте:

1. Нажмите "Generate new token" → "Generate new token (classic)"
2. Заполните:
   - **Note:** `Hampfree Blog - Auto Auth`
   - **Expiration:** 90 дней
   - **Scopes:** только `repo` ✅
3. Нажмите "Generate token"
4. **Скопируйте токен** (показывается один раз!)
