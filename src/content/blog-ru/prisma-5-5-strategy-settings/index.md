**Prisma 5.5** — авторская стратегия платформы [Veles](https://veles.finance/invite/washmallay) для автоматической торговли. В этой статье — полный разбор настроек: какие индикаторы используются для входа, как выставить сетку ордеров и на каких парах запускать.

**Важно:** перед использованием любой стратегии ознакомьтесь с [управлением рисками](/ru/library/risk-management-crypto-trading-bots/) и серией [Диверсификация и настройка торговой системы](/ru/library/diversification-trading-system-part-1/).

## Что такое Prisma 5.5

Prisma 5.5 — это высокочастотная торговая стратегия, разработанная для работы на волатильном крипторынке. Она сочетает в себе фильтры тренда и осцилляторы для поиска оптимальных точек входа в Long позиции.

## Настройки стратегии

### Основные параметры

| Параметр | Значение |
|---|---|
| Направление | Long |
| Биржа | Bybit / BingX / Bitget |
| Сетка ордеров | 25-30 ордеров |
| Перекрытие | 15-20% |
| Отступ первого ордера | 0.2% |
| Мартингейл | 1.05 - 1.1 |

### Индикаторы входа (Фильтры)

Для открытия позиции в Prisma 5.5 используются следующие индикаторы:

1. **CCI (Commodity Channel Index)**
   * • Период: 14
   * • Уровень: -100
   * • Условие: CCI меньше -100 (зона перепроданности)

2. **Bollinger Bands**
   * • Период: 20, Отклонение: 2
   * • Условие: Цена ниже нижней полосы

3. **RSI (Relative Strength Index)**
   * • Период: 14
   * • Уровень: 30
   * • Условие: RSI меньше 30

### Управление сеткой ордеров

Сетка настраивается с учетом текущей волатильности:
* • **Шаг ордеров:** логарифмический, начиная с 0.5%
* • **Коэффициент мартингейла:** позволяет быстрее выходить в плюс при развороте цены.

## На каких парах запускать

Рекомендуется использовать ликвидные пары с высоким объемом торгов:
* • **TOP-10:** BTC/USDT, ETH/USDT, SOL/USDT.
* • **Волатильные альткоины:** ARB/USDT, OP/USDT, LINK/USDT.

## Как запустить Prisma 5.5 на Veles

1. Зарегистрируйтесь на [Veles](https://veles.finance/invite/washmallay) и привяжите API-ключ биржи.
2. Перейдите в раздел **«Готовые боты»** и выберите фильтр по автору или названию Prisma 5.5.
3. Нажмите **«Копировать»**, выберите торговую пару и укажите выделенный депозит.
4. Проверьте параметры через кнопку «Рассчитать» и нажмите **«Запустить»**.

## Заключение

Prisma 5.5 — это сбалансированная стратегия для тех, кто хочет автоматизировать торговлю на споте или фьючерсах с умеренным риском. Всегда следите за уровнем маржи и не забывайте про диверсификацию.
