## Что такое Veles

**Veles Finance** — это SaaS-платформа (Software as a Service), которая предоставляет готовые инструменты для алгоритмической торговли. Не нужно устанавливать ПО, настраивать серверы или писать код — всё работает в облаке.

**Основная идея:** Сделать алготрейдинг доступным для розничных трейдеров с небольшим капиталом ($100-10,000).

### Ключевые возможности

1. **Grid-боты** — сетка ордеров на покупку/продажу в заданном диапазоне
2. **DCA-боты** — усреднение позиции по расписанию или сигналу
3. **Фьючерсные боты** — торговля с плечом (до 20x)
4. **Бэктесты** — тестирование стратегий на исторических данных
5. **Готовые стратегии** — библиотека настроек от сообщества
6. **Маркетплейс** — покупка/продажа авторских стратегий

## Как работает Veles

### Архитектура

```
Вы → Veles (облако) → Биржа (API) → Исполнение ордеров
      ↓
   Бэктесты, статистика, уведомления
```

1. **Регистрация:** Создаёте аккаунт на Veles
2. **Подключение биржи:** Добавляете API-ключи (только торговля, без вывода!)
3. **Настройка бота:** Выбираете стратегию, параметры, депозит
4. **Запуск:** Бот начинает торговать 24/7
5. **Мониторинг:** Следите через веб-интерфейс или Telegram-бот

### Типы ботов

#### 1. Grid-бот (сетка)

**Принцип:** Покупает дешево, продает дорого в заданном диапазоне.

**Настройки:**
- Диапазон цены (мин/макс)
- Количество сеток (5-100)
- Размер позиции на сетку
- Stop-loss (опционально)

**Пример:**
```
Диапазон: $60,000 - $70,000 (BTC)
Сеток: 10
Размер: $100 на сетку

Цена $65,000 → Бот покупает на $100
Цена $66,000 → Бот продаёт, фиксирует прибыль
Цена $64,000 → Бот покупает снова
```

**Для кого:** Боковой рынок (флэт), высокая волатильность.

#### 2. DCA-бот (усреднение)

**Принцип:** Покупает через равные интервалы или при падении цены.

**Настройки:**
- Базовая сумма покупки
- Интервал (время или % падения)
- Множитель (увеличение покупки при падении)
- Take-profit (когда фиксировать прибыль)

**Пример:**
```
Базовая покупка: $100
Интервал: Каждые 24 часа
Множитель: 1.5x при падении на 5%

День 1: Купил на $100
День 2: Цена упала на 5% → Купил на $150
День 3: Цена упала ещё на 5% → Купил на $225
Цена выросла на 10% → Продал всё с прибылью
```

**Для кого:** Долгосрочные инвестиции, накопление позиции.

#### 3. Фьючерсный бот

**Принцип:** Торговля с плечом (лонг/шорт).

**Настройки:**
- Направление (лонг/шорт)
- Плечо (2-20x)
- Размер позиции
- Stop-loss / Take-profit

**Риски:** Высокие! Можно потерять больше депозита при неправильном управлении рисками.

**Для кого:** Опытные трейдеры, хеджирование позиций.

## Личный опыт: Настройка Grid-бота на BTC

**Дата теста:** 20 марта 2026  
**Биржа:** Bybit  
**Пара:** BTC/USDT  
**Стратегия:** Grid (нейтральная)  
**Депозит:** $100

### Шаг 1: Подготовка

**Регистрация:** 5 минут (email + пароль)  
**Верификация:** Не требуется (для сумм до $10,000)  
**Подключение Bybit:** 2 минуты (создание API-ключей)

### Шаг 2: Настройка бота

**Интерфейс Veles:**
1. Выбрал пару: BTC/USDT
2. Тип бота: Grid (нейтральный)
3. Диапазон: $60,000 - $70,000
4. Количество сеток: 20
5. Депозит: $100

**Расчёт параметров:**
- Размер сетки: $500 ($70,000 - $60,000 / 20)
- Покупка на сетку: ~$5
- Прибыль на сетку: ~0.5-1%

### Шаг 3: Запуск

**Время запуска:** 14:30 МСК  
**Цена BTC:** $64,200  
**Первые ордера:** 
- Buy: $64,000, $63,500, $63,000...
- Sell: $64,500, $65,000, $65,500...

### Шаг 4: Результаты (7 дней)

**Статистика:**
- Количество сделок: 47
- Прибыль от сеток: $3.20
- Unrealized PnL (плавающий убыток): -$1.80
- **Чистая прибыль:** $1.40 (1.4% за неделю)

**APY (годовых):** ~73% (если бы рынок был идеально боковым)

**Реальность:**
- 3 дня рынок рос → бот продавал и оставался без позиции
- 2 дня рынок падал → бот покупал, но не успевал продать
- 2 дня был флэт → бот отработал идеально

### Выводы

**Плюсы:**
- ✅ Работает 24/7 без участия
- ✅ Зарабатывает на волатильности
- ✅ Прозрачная статистика
- ✅ Можно остановить в любой момент

**Минусы:**
- ❌ Неэффективно в тренде (продаёт рано или покупает падающее)
- ❌ Комиссии биржи съедают прибыль на малых суммах
- ❌ Нужен постоянный мониторинг и перенастройка

**Рекомендация:** Grid-боты работают лучше всего на боковом рынке с высокой волатильностью. Для трендового рынка — использовать DCA или лонг/шорт стратегии.

## FAQ

**Безопасно ли доверять API-ключи Veles?**

Да, если выдать ключи только на торговлю (без вывода). Veles не может вывести средства — только торговать.

**Сколько нужно денег для старта?**

Минимум: $50-100 (но комиссии съедят прибыль).  
Оптимально: $500-1,000 для спота, $200-500 для фьючерсов.

**Можно ли потерять больше депозита?**

На споте — нет (максимум -100% от позиции).  
На фьючерсах — да, при торговле с плечом и отсутствии stop-loss.

**Работает ли Veles в России?**

Да, Veles доступен в РФ. Биржи (Bybit, Binance) тоже работают с ограничениями.

**Нужно ли платить налоги?**

Да, прибыль от торговли криптовалютой облагается налогом (13% для резидентов РФ). Veles предоставляет экспорт для учёта.
