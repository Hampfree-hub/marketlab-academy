**Ginarea** — платформа для создания торговых ботов с конструктором стратегий. В этой статье — подробная настройка каждого типа бота, реальный опыт работы с платформой и конкретные примеры конфигураций. Для общего понимания возможностей платформы рекомендуем [обзор Ginarea](/ru/library/ginarea-platform-overview/), а для знакомства с основами автоматизации — [гайд по торговым ботам](/ru/library/trading-bots-guide-2026/). Метрики для оценки активов, упомянутые в контексте выбора монет, разобраны во [введении в фундаментальный анализ](/ru/library/fundamental-analysis-intro/).

## История работы с платформой

Знакомство с Ginarea состоялось несколько лет назад, но плотное возвращение к платформе произошло только в 2026 году. Для тестирования был выделен скромный депозит — около **$300 на одном из субаккаунтов Bybit** (после [ликвидации одного из счетов](https://t.me/hampfree_market_lab/26)). Позже, с расширением направления, депозит пополнили ещё примерно на $100.

На BitMEX тестирование проходило на совсем небольшом депозите — **$50–60**.

Для того чтобы освежить память и плавно войти в торговлю на каждой из бирж, начинали с минимального количества активов и минимально допустимого размера ордера. Только после калибровки настроек добавляли новые активы в торговую систему, внимательно отслеживая уровень начальной маржи.

## Подготовка: подключение биржи

⚠️ **Важно:** Перед настройкой бота нужно подключить API ключи биржи.

1. Зарегистрируйтесь на бирже (Bybit, OKX, KuCoin, BitMEX)
2. Создайте API ключи в настройках аккаунта
3. В Ginarea перейдите в раздел «Биржи» → «Добавить биржу»
4. Вставьте API Key и Secret Key
5. Проверьте подключение (кнопка «Тест»)

## Типы ботов в Ginarea

**Доступные типы:**
1. **Default** — базовый бот для простой торговли
2. **Auto Grid** — автоматическая сетка с адаптацией к цене
3. **DCA** — усреднение стоимости позиции
4. **Indicator Grid** — сетка + индикаторы для фильтрации сигналов
5. **Dynamic** — динамическая перестройка параметров

## 1. Default бот: базовая настройка

**Для кого:** Новички, которые хотят попробовать автоматизацию.

**Шаги настройки:**

1. Перейдите в раздел «Боты» → «Создать бота» → «Default»
2. Выберите биржу и торговую пару (например, BTC/USDT)
3. Настройте параметры:
   - **Сумма ордера:** минимальная сумма на бирже (например, $10)
   - **Направление:** Long / Short / Neutral
   - **Стоп-лосс:** % убытка для автоматического закрытия
   - **Тейк-профит:** % прибыли для фиксации
4. Запустите бота и наблюдайте за статистикой

## 2. Auto Grid: автоматическая сетка

**Для кого:** Трейдеры, которые торгуют в боковике.

**Шаги настройки:**

1. Перейдите в раздел «Боты» → «Создать бота» → «Auto Grid»
2. Выберите биржу и пару
3. Настройте параметры:
   - **Диапазон цены:** мин. и макс. цена для сетки
   - **Количество уровней:** сколько ордеров в сетке (5–20)
   - **Сумма на уровень:** сумма каждого ордера
   - **Автоперестройка:** вкл/выкл (адаптация к движению цены)
4. Запустите бота

## 3. DCA: усреднение стоимости

**Для кого:** Консервативные инвесторы для накопления на длинной дистанции.

**Шаги настройки:**

1. Перейдите в раздел «Боты» → «Создать бота» → «DCA»
2. Выберите биржу и пару
3. Настройте параметры:
   - **Сумма покупки:** фиксированная сумма (например, $50)
   - **Интервал:** как часто покупать (1 час, 4 часа, 1 день, 1 неделя)
   - **Направление:** Long / Short
   - **Условие остановки:** когда прекратить покупки
4. Запустите бота

## 4. Indicator Grid: сетка + индикаторы

**Для кого:** Опытные трейдеры, которые используют технический анализ.

**Шаги настройки:**

1. Перейдите в раздел «Боты» → «Создать бота» → «Indicator Grid»
2. Выберите биржу и пару
3. Настройте параметры:
   - **Диапазон цены:** мин. и макс. цена
   - **Количество уровней:** 5–20
   - **Индикатор:** RSI, MACD, Bollinger Bands
   - **Условие входа:** когда открывать позиции (например, RSI < 30)
   - **Условие выхода:** когда закрывать позиции (например, RSI > 70)
4. Запустите бота

## 5. Dynamic: динамическая перестройка

**Для кого:** Продвинутые пользователи для адаптации к рыночным условиям.

**Шаги настройки:**

1. Перейдите в раздел «Боты» → «Создать бота» → «Dynamic»
2. Выберите биржу и пару
3. Настройте параметры:
   - **Базовая стратегия:** Default / Grid / DCA
   - **Параметры адаптации:** как менять параметры при движении цены
   - **Частота пересчёта:** как часто пересчитывать параметры
4. Запустите бота

## P&L Трейлинг: критически важная настройка

Первое время настоятельно рекомендуется внимательно изучить [настройки P&L трейлинга](https://ginareas-organization.gitbook.io/ginarea.org/nastroiki-botov/nastroiki-p-and-l-treilinga) в официальной документации.

**Стартовая конфигурация для текущего рынка:**

- **Дистанция:** 4%
- **% на трейлинг:** 60%

Это означает, что при отдалении от текущей цены более чем на 4% — **60% профита** с закрытых ордеров внутри позиции будет отбрасываться на погашение повисших ордеров вне диапазона.

Это не панацея, но данный гибкий подход позволяет довольно быстро и удобно контролировать открытую позицию. В критические моменты выставляли конфигурацию **1%/90%** — пару раз это реально спасало от глубокой просадки.

## Аналитика и статистика

**Где смотреть:**

1. Перейдите в раздел «Аналитика»
2. Выберите бота из списка
3. Просмотрите метрики:
   - **Общая прибыль:** % и $
   - **Количество сделок:** всего и за период
   - **Средний чек:** средняя прибыль/убыток на сделку
   - **Просадка:** максимальное падение от пика
   - **Время работы:** сколько бот активен

## Частые ошибки при настройке

### Ошибка 1: Слишком большой диапазон сетки

**Проблема:** Сетка не срабатывает, цена не доходит до уровней.

**Решение:** Уменьшите диапазон, используйте Auto Grid с автоперестройкой.

### Ошибка 2: Маленькая сумма ордера

**Проблема:** Биржа отклоняет ордера (ниже минимума).

**Решение:** Проверьте минимальную сумму ордера на бирже и настройте бота соответственно.

### Ошибка 3: Нет стоп-лосса

**Проблема:** При сильном движении против бота — большие убытки.

**Решение:** Всегда ставьте стоп-лосс (5–10% от депозита).

### Ошибка 4: Слишком много активов одновременно

**Проблема:** Начали торговать 10 активами с минимального депозита — маржа распределяется слишком тонко.

**Решение:** Начинайте с 1–3 активов, постепенно добавляя новые после калибровки настроек. Отслеживайте уровень начальной маржи.

## Итоги

**Главное:**
1. Default — для новичков (простая настройка)
2. Auto Grid — для боковика (автоматическая адаптация)
3. DCA — для накопления (усреднение на длинной дистанции)
4. Indicator Grid — для опытных (сетка + индикаторы)
5. Dynamic — для продвинутых (динамическая перестройка)

**Ключевой принцип:** Начинайте с минимальных сумм, калибруйте настройки на 1–2 активах, затем масштабируйтесь. P&L трейлинг — ваш лучший друг для контроля позиции.

## FAQ

**Сколько стоит использование Ginarea?**

Ginarea бесплатна для использования. Платформа зарабатывает на партнёрской программе (комиссия от бирж).

**Какие биржи поддерживаются?**

OKX, Bybit, BitMEX, KuCoin. Полный список доступен в разделе «Биржи» после подключения.

**Нужно ли программирование для настройки бота?**

Нет. Ginarea — визуальный конструктор, настройка происходит через интерфейс без написания кода.

**Можно ли запустить несколько ботов одновременно?**

Да. Количество ботов не ограничено. Рекомендуется начинать с 1–2 ботов для тестирования.

**Как вывести прибыль от бота?**

Прибыль остаётся на бирже. Выводите через интерфейс биржи (раздел «Кошелёк» → «Вывод»).

**Что делать, если бот убыточный?**

Проверьте настройки: диапазон сетки, сумму ордера, стоп-лосс, P&L трейлинг. При необходимости остановите бота и скорректируйте параметры.

**Где посмотреть официальные инструкции по настройке?**

Полная документация доступна на [GitBook Ginarea](https://ginareas-organization.gitbook.io/ginarea.org/nastroiki-botov).

**Какой минимальный депозит нужен для начала?**

Можно начать с $50–100 на тестовом аккаунте. Главное — не распыляться на множество активов и начинать с минимальных ордеров.
