Файлы в public/ — что зачем, коммитить ли

● robots.txt
  Назначение: Разрешает индексацию (Allow: /) и указывает sitemap.
  Нужен: Да, для SEO. Поисковики читают его с корня сайта.
  Зелёный в git: Новый/не закоммичен — коммитить.

● GOOGLE_SEARCH_CONSOLE
  Инструкция по проверке владения: HampfreeBlog-Private/docs/GOOGLE_SEARCH_CONSOLE_README.txt.
  В public/ её нет — только в приватном репо.

● googleXXXXXXXX.html (если появится)
  Скачанный из Search Console файл проверки. Положить сюда, закоммитить,
  после проверки можно удалить или оставить.
