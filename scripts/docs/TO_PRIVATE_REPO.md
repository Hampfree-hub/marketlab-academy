# Список файлов для приватного репо

Список путей и процедур синхронизации хранится **только в приватном репозитории** (HampfreeBlog-Private), чтобы в публичный репо не попадали внутренние пути и данные.

См. в приватном репо: `docs/TO_PRIVATE_REPO.md` или раздел «Синхронизация из публичного репо» в `README.md`.
