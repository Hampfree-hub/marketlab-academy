# Публикации по источникам

Краткие посты для соцсетей и каналов (Telegram, Дзен, Paragraph и др.). Объём — до ~2000 символов. Здесь хранятся черновики и финальные тексты перед публикацией; в блоге выходят полноценные статьи.

**Соглашение:**
- Один файл — один пост или одна кампания (по источнику/дате).
- В имени файла: slug стратегии или темы + источник при необходимости, например `oracle-1-1-telegram-dzen.md`.

**Статьи с настройками в блоге:** в конце каждой такой статьи — приглашение посетить страницу [О проекте](https://marketlab-academy.org/ru/about/) (то же для EN/ES).
